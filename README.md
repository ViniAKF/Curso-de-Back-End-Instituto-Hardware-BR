# Curso-de-Back-End-Instituto-Hardware-BR
Olá! Seja bem vindo! Este é o repositório referente ao curso de Back-End oferecido pelo Instituto Hardware BR em parceria com a Fundação Liberato!
O principal foco do curso é na linguagem de programação Python e, apesar de já ter trabalhado com a linguagem, vão ser postados códigos acompanhando toda linha cronológica do curso, desde os exercícios iniciais.

Pastas:
- AnotacoesDasAulas     ->  Anotações feitas durante as aulas
 
- ExerciciosIniciais    ->  Exercícios desenvolvidos com o intuito de trabalhar com as funções básicas da programação, se adequando à sintaxe do Python e seus diferenciais.
